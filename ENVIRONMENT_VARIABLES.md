# Environment variables

**Both apps**

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY`

**kleen-app**

- `NEXT_PUBLIC_SITE_URL` (customer app URL, e.g. `https://dashboard.kleenapp.co.uk` — drives booking/sign-in links from www)
- `NEXT_PUBLIC_AUTH_COOKIE_DOMAIN` (optional, e.g. `.kleenapp.co.uk` for shared auth across subdomains)
- `NEXT_PUBLIC_MARKETING_URL`
- `NEXT_PUBLIC_ENABLE_EMAIL_AUTH`
- `NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY`
- `STRIPE_SECRET_KEY`
- `STRIPE_WEBHOOK_SECRET`
- `RESEND_API_KEY`
- `RESEND_FROM_EMAIL`
- `RESEND_FROM_VERIFIED`
- `RESEND_REPLY_TO`
- `RESEND_FORCE_ONBOARDING`
- `ADMIN_NOTIFY_EMAIL`
- `ADMIN_APP_URL`
- `CRON_SECRET`
- `DATABASE_URL`
- `VERCEL_URL` (Vercel injects)

**kleen-admin**

- `NEXT_PUBLIC_CUSTOMER_APP_URL`
- `STRIPE_SECRET_KEY`
- `RESEND_API_KEY`
- `RESEND_FROM_EMAIL`
- `RESEND_FROM_VERIFIED`
- `CUSTOMER_DASHBOARD_URL`
- `CONTRACTOR_PORTAL_BASE_URL`
